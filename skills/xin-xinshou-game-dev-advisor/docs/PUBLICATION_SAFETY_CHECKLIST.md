# 公开发布安全检查表

发布前逐项确认。

## Secret / 隐私

- [ ] 没有 `.env`。
- [ ] 没有 API key。
- [ ] 没有 GitHub token。
- [ ] 没有 OpenAI key。
- [ ] 没有 cookie / session。
- [ ] 没有本地账号路径。
- [ ] 没有私人邮箱或手机号。

## 版权 / 授权

- [ ] 没有未授权字体。
- [ ] 没有未授权图片。
- [ ] 没有复制大段书籍、小说、论文或网页原文。
- [ ] 没有使用知名武侠 IP 名称、门派、人物、标志。
- [ ] LICENSE 文件存在。

## 项目边界

- [ ] 只包含 Skill、说明、模板。
- [ ] 不包含真实游戏源码。
- [ ] 不包含私有 PR、issue、仓库内部状态。
- [ ] 不包含构建产物。
- [ ] 不包含截图 artifact。

## AI 资产

- [ ] 没有 AI 原始候选图。
- [ ] 没有将 AI 图标记为最终可商用资产。
- [ ] 若以后加入资产，必须有 AI_ASSET_LEDGER。

## GitHub 设置

- [ ] 仓库名清楚。
- [ ] 仓库是 public，确认用户愿意公开。
- [ ] README 首屏说明准确。
- [ ] Release tag 创建成功。
- [ ] 默认分支无敏感内容。
