# 使用方法

## 1. 启用方式

把 `SKILL.md` 放进你的项目技能库、项目说明文档或 AI 助手上下文，然后输入：

```text
启用「新新手游戏开发顾问」。
请按新手项目掌门视角帮我判断。
```

## 2. 常用触发语

### 项目体检

```text
我是新手，现在项目有点乱，请用「新新手游戏开发顾问」做一次项目体检。
```

### 写 Codex 任务卡

```text
请用「新新手游戏开发顾问」帮我把这个需求拆成 Codex 任务卡。
需求：<粘贴需求>
```

### 审查 PR

```text
请用「新新手游戏开发顾问」审查这个 PR 是否可以合并。
材料：<PR body / changed files / CI / screenshots / build report>
```

### 每日同步

```text
开始 YQJH Daily Sync。
以下是今天的聊天、PR、点子、阻塞和截图反馈。
```

### GitHub 公开发布准备

```text
请用「新新手游戏开发顾问」检查这个 skill 是否可以发布为 GitHub public repo。
```

## 3. 推荐回答风格

顾问输出应保持：

```text
判断明确
术语解释给新手听
下一步可以直接执行
必须给 Codex 任务卡
必须标出 Human Gate
```

## 4. 不适合让它做什么

不要让此 Skill：

```text
直接决定 merge
直接决定 Steam 合规
直接决定版权安全
直接把 AI 图当最终资产
直接替代人工视觉签收
直接公开发布私有项目内容
```
